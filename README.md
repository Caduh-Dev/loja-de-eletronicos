# Projeto Banco de Dados – Loja de Eletrônicos

Projeto desenvolvido em PostgreSQL + PGAdmin contendo criação de tabelas, inserção de dados, consultas SQL e comandos de atualização e remoção.

## Estrutura
- 01_create_tables.sql
- 02_insert_data.sql
- 03_select_queries.sql
- 04_update_data.sql
- 05_delete_data.sql

## Execução
Execute os scripts na ordem acima em um banco chamado `loja_eletronicos`.
